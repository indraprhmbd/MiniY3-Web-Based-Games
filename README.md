# MiniY3 - Next.js Game Folder

This directory contains the source code for the MiniY3 platform.

## Key Files

- `app/page.tsx`: Landing page with game grid.
- `app/games/lucky-duel/page.tsx`: Local mode game.
- `app/games/lucky-duel-online/page.tsx`: Online mode game.
- `lib/supabase.ts`: Supabase client configuration.

Refer to the root `README.md` for full instructions on how to setup and run the project.
